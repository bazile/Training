Примеры реализации интерфейсов IEnumerable/IEnumerator

01-Vector
	Самый простой вариант реализации IEnumerable в котором используется уже готовая реализация для массивов

02-MyList
	Необобщенный класс MyList с реализацией IEnumerable/IEnumerator в виде вложенного класса

03-MyList-Generic
	Обобщенный класс MyList<T> с реализацией IEnumerable<T>/IEnumerator<T> в виде вложенного класса

04-MyList-Generic-yield
	Обобщенный класс MyList<T> с реализацией IEnumerable<T>/IEnumerator<T> с помощью yield
